Overview

DePo Clean is the default theme for Melody 1.0. It is derived from a WordPress theme by the same name by Derek Powazek. It is a minimalist theme intended to give a sharp and clean display of your content. By default, it publishes a main page, an atom feed, individual entry archives, comment feeds and listing archives (monthly, category and author by default). It comes with several widgets.

Theme Options

  • Flickr
  • Listing Widgets
  • Per Page Entry Limits
  • User and Site Metadata

Flickr

In order to use the provided Flickr support, you will need to get your Flickr ID. Go here to get yours if you don't have know what it is. The Flickr module supports several variations. For all of them you will need to have your Flickr ID in order for Flickr to let you interface with it. For a group gallery, you will also need to have the group's ID code as well.

Listing Widgets

This section allows you to control how many items will be displayed in three of the widgets:

  • Monthly archives
  • Recent comments
  • Recent entries

Per Page Entry Limits

The purpose of this section is to allow fine-grained control of how many entries are displayed on each of the page types which display more than one entry.

User and Site Metadata

This section allows you to enter text to provide additional information about your blog, contact information and set up the Action Streams plugin.
